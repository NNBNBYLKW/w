# 测试版发布准备

## 1. 当前 beta 发布目标

本文件说明 `Workbench Beta 0.1.0` 的当前交付形态、验证边界和已知限制。

当前推荐交付物是 **一体化 Windows beta 包**：用户安装后双击启动，不需要手动启动 Python backend，也不需要自行配置 FFmpeg。

这仍然是 beta 发布准备文档，不是正式商业发布说明，也不是新功能规划。

## 2. 当前 beta 包包含什么

当前一体化 Windows beta 包包含：

- Electron 桌面壳
- 已构建的 frontend 静态资源
- PyInstaller `onedir` 打包的 backend executable
- 随包携带的 `ffmpeg.exe`
- 桌面 bridge：选择文件夹、打开文件、打开所在文件夹、窗口控制

当前 beta 包不包含：

- 自动更新系统
- 后台服务安装或系统级自启动
- 复杂安装向导
- 正式发布级应用图标 / 签名收口
- SQLite 初始用户数据

旧的技术 beta 包已归档到：

```text
apps/desktop/release/archive/technical-beta-0.1.0-manual-backend-20260425/
```

该旧包需要用户手动启动 backend，仅作为回退参考，不再作为推荐交付形态。

## 3. 安装 / 启动前提

面向非技术用户的一体化包预期行为：

- 安装后双击 `Workbench Beta`
- Electron 自动启动本地 backend
- backend ready 后进入 frontend
- 应用退出时关闭本次启动的 backend 子进程
- 视频缩略图与 DetailsPanel 6 帧预览默认使用随包携带的 FFmpeg

packaged mode 的本地 backend 默认运行在：

```text
http://127.0.0.1:8765
```

开发模式仍可直接运行 backend：

```powershell
cd apps/backend
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

## 4. 数据目录

一体化包不把正式运行数据写回仓库目录。

packaged mode 下运行数据写入 Electron `userData` 目录，例如：

```text
%APPDATA%/Workbench Beta/backend-data/
```

其中包含：

- `workbench.db`
- `thumbnails/`
- `thumbnails/video/`
- `thumbnails/video_preview/`

开发模式仍默认使用：

```text
apps/backend/data/
```

## 5. 当前打包命令

```powershell
cd apps/desktop
npm run package:win
```

该命令会依次完成：

- frontend production build
- backend PyInstaller build
- bundled FFmpeg resource prepare
- desktop TypeScript build
- electron-builder Windows NSIS package

Windows 一体化包输出目录：

```text
apps/desktop/release/integrated/
```

当前安装包名：

```text
Workbench Beta Integrated Setup 0.1.0.exe
```

## 6. 建议 smoke test

发布前至少检查：

- 不手动启动 Python backend，直接启动 `win-unpacked/Workbench Beta.exe`
- `/health` 在 `http://127.0.0.1:8765` 可访问
- 无 saved source 时进入 `Onboarding`
- 有 saved source 时进入 `Home`
- 左侧导航可进入 Search / Files / Recent / Media / Games / Books / Software / Tags / Collections / Settings
- 单击条目能刷新 shared details
- 双击条目或 details open actions 在 Electron 中不崩溃
- tags / color tags / favorite / rating 可写入与回显
- Recent batch organize 可进入、操作后可再找回
- Collections / Tags / Recent family 的回流入口可用
- Settings 语言切换可用
- 图片缩略图可用
- 视频缩略图可用
- DetailsPanel 的视频 6 帧循环预览只在视频详情中出现
- 关闭应用后 backend 子进程退出；再次启动不端口冲突

## 7. 已知限制

- 当前仍是 beta，不是正式版发行。
- backend 是随桌面包启动的本地子进程，不安装为系统服务。
- FFmpeg 来自随包携带的第三方二进制，发布时需要保留对应许可说明。
- 视频多帧预览只在 DetailsPanel 中出现，不在列表、hover 或卡片中出现。
- Collections 仍是 saved retrieval surface，不是规则引擎。
- Games status 仍是 Games 专属语义，不是全站统一 status。
- 当前不支持 AI 自动标签、OCR、推荐系统或自动规则。

## 8. 反馈回收建议

外部反馈建议按下面优先级记录：

- P0：无法启动、关键页面崩溃、主路径无法完成、backend 子进程残留导致无法重启
- P1：主路径明显卡住、文案误导严重、桌面动作不可用、视频预览在常见视频上异常
- P2：可带入 beta 的体验瑕疵、低可见文案或视觉问题
- P3：未来功能期待或非当前范围建议

反馈时优先记录：

- 用户使用的 Windows 版本
- 是否从一体化安装包启动
- source 类型与样本文件类型
- 具体在哪个页面、哪个动作出现问题
- 是否能稳定复现
