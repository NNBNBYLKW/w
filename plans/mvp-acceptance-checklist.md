# MVP Acceptance Checklist

- user can add a source
- user can trigger a scan flow
- files can later be indexed from that source
- search can later find indexed files
- details panel can later inspect a selected file
- tags and color tags can later be applied
- media library can later browse indexed image or video files
- recent imports can later act as an organization entry point
- app shell remains shared across pages
- project does not drift into Explorer replacement or platform sprawl
